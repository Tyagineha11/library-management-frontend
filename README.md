Library Management System - Frontend

Overview:
The Library Management System Frontend is a modern web application built with React and TypeScript. It provides an intuitive and responsive user interface for managing library operations such as books, members, book issuing, returns, fines, and user authentication. The application communicates with the Spring Boot backend through REST APIs.

Features:
* Secure Login & Authentication
* Dashboard
* Book Management
* Member Management
* Issue & Return Books
* Fine Management
* Search & Filter
* Responsive User Interface
* Form Validation
* API Integration

Tech Stack:
* React
* TypeScript
* Vite
* React Router
* Axios
* CSS / Tailwind CSS (if used)
* React Icons

Getting Started:
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
    ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open the application in your browser.
